# Intune Automation

Coleção de scripts PowerShell para automatizar o empacotamento, instalação e detecção de aplicativos via Microsoft Intune (Win32 apps).

## 🎯 Objetivo

Reduzir trabalho manual repetitivo na implantação de softwares corporativos, padronizando o processo de:
- Empacotamento de aplicativos no formato `.intunewin`
- Instalação silenciosa via script
- Detecção de instalação (regras customizadas)
